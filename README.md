# 密碼產生器
![封面截圖](/public/image/index.png)
## 介紹
可以依個人需求產生出隨機密碼。
## 功能
- 可以選擇需要的密碼長度、需要的字或符號
- 產出密碼後，會記憶上次選擇的規格
- 可以利用Copy按鈕來複製
## 如何使用
1. 在terminal中clone此專案：
```
git clone https://github.com/Yuanuanuan/password_generator.git
```
3. 在terminal中移動到此檔案後輸入：
```
npm install
```
4. 安裝完畢後，輸入：
```
npm run start
```
5. 若顯示以下訊息代表成功運行：
```
伺服器已運作，監聽在 http://localhost:3000
```
6. 將網址打開即可開始使用
```
http://localhost:3000
```
8. 若要關閉，按以下即可
```
Crtl+C
```
